How it works

Scramble asks the server for a random sequence of moves and animates them on the cube.

Solve sends the current 54-sticker state to the server, which runs Batch-Weighted A\* with the trained neural heuristic to find a move sequence back to solved. λ trades optimality for speed (1 = optimal A*, higher = greedier).
